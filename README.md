# ceftolozane-tazobactam

Population pharmacokinetics of ceftolozane–tazobactam using classical NONMEM and Bayesian approaches

This repository contains the electronic version that forms the basis of the article “Nonlinear Mixed-Effects and Full Bayesian Population Pharmacokinetic Analysis of Ceftolozane–Tazobactam in Critically Ill Patients.”
